Senior Software Engineer • March, 2015 — Present

MobileCause

Senior Software Engineer & Architect • April, 2012 — March, 2015

At MobileCause, I worked with a small engineering team to help build a mobile fundraising platform that helps non-profit organizations raise money effectively.

  • Worked on an SMS message based fundraising platform that included 3 Ruby on Rails applications & a Java based messaging aggregator & router.
  • Worked primarily on the Rails application that serves as the administrative interface for the SMS messaging platform.   • Wrote & open sourced an API integration wrapper for the CardConnect API in Ruby. See Projects section below for more details.
